Oh My Vim
=========
Oh My Vim 是一套开箱即可用的 Vim 配置
![截图](https://raw.githubusercontent.com/gavin-hu/oh-my-vim/master/oh-my-vim.png)

操作系统
========
1. Mac OSX
2. Linux     （未测试）

环境准备
========
1. curl
2. git
3. ctags (非必须)

一键安装
========
curl https://raw.githubusercontent.com/gavin-hu/oh-my-vim/master/bootstrap.sh -L -o  - | sh
